# Dscan version 1.2
     Demons Scanner
A tool designed to scan the open ports on any host


installer:
  

     git clone https://github.com/demonsteam/DPscan.git
     cd dscan
     chmod +x install
     ./install
     dscan --help





   Usage:


        dscan -r <target host>
        dscan -u <target host>








example:



        dscan -r 192.168.1.1
        dscan -u google.com
